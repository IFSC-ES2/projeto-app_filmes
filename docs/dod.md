Definition of Done (dod) - CineLog 

1. Código: Segue o padrão de nomenclatura (CamelCase para Java).
2. Revisão: Pelo menos 1 aprovação de outro membro no Pull Request.
3. Testes: Critérios de aceitação validados manualmente ou via JUnit.
4. Documentação: README ou ADR atualizados se a arquitetura mudou.
5. Build: O projeto precisa compilar sem erros.
