# ADR-0001 - Java com Spring Boot

### Status
Aceita

## Contexto


## Decisão
A equipe decidiu utilizar Java com Spring Boot no backend e MySQL
como banco de dados principal.

## Alternativas consideradas
- React 
- PostgreSQL 

## Consequências
### Positivas
- Boa organização para aplicações em camadas
- Facilidade para testes e manutenção

### Negativas / trade-offs
- Maior configuração inicial em comparação a opções mais simples
- Não funcionar tudo que queremos em pouco tempo.

## Revisão futura
A decisão da equipe poderá ser revista caso surjam restrições técnicas relevantes
ou dificuldades excessivas de implementação.
