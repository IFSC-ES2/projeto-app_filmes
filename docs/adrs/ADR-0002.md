# ADR-0002 - Estática para React

### Status
Aceita

## Contexto
Com a avaliação do nosso cliente, resolvemos alterar o Frontend de Estático para React. Vai nós facilitar o desenvolvimento do Frontend dévido as múltiplas páginas planejadas.

## Decisão
A equipe decidiu utilizar React invés de Estático no Frontend.

## Alternativas consideradas
Alternativas não foram consideradas.

## Consequências
### Positivas
- Componentização e Reutilização.
- Gerenciamento de Estado Dinâmico.
- Padronização de Código.

### Negativas / trade-offs
- Complexidade e Curva de Aprendizado.
- Sobrecarga para Sites Simples.

## Revisão futura
A decisão da equipe poderá ser revista caso surjam restrições técnicas relevantes
ou dificuldades excessivas de implementação.
