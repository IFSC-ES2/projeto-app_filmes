# Planejamento Atualizado do projeto

## 1. Recorte do Backlog (Escopo do MVP)

O baseline do projeto CineLog foi estabelecido com foco estrito no desenvolvimento do MVP. O escopo inicial mapeia as seguintes entregas funcionais centrais:

* Cadastro de títulos (filmes ou séries).

* Atribuição de nota/avaliação (1 a 5).

* Listagem de títulos avaliados.

* Exclusão de registros salvos.
